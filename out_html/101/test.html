問題文

ただし、この男性の健常時の体重は70kg、血清ナトリウム濃度の目標値を140mEq/L、体水分量を体重の60%とする。

  1. 1.0L
  2. 1.5L
  3. 2.0L
  4. 2.5L
  5. 3.0L

解答

3

解説

健常時であれば、70kg の 60 % = 42 kg = 42L が体内の水分です。※ 水 1 L が、1kg は、知識です。mEq/L というのは、mmol/L と考えればOKです。

42 L で、140mEq/L が普通で、水が減って、147mEq/L となった。ということになります。つまり濃度が 5 % 増えています。すると、水分が大体 5 % 減った　と考えればよいです。42 L の 5 % は、2.1 L なので、一番近い選択肢は 3 です。

以上より、正解は 3 です。
