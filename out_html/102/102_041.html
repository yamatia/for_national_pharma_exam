　問題文　

薬物の腸肝循環の経路はどれか。1つ選べ。

  1. 肝臓 → 門脈 → 胆管 → 腸管 → 肝臓
  2. 肝臓 → 門脈 → 腸管 → 胆管 → 肝臓
  3. 肝臓 → 胆管 → 門脈 → 腸管 → 肝臓
  4. 肝臓 → 胆管 → 腸管 → 門脈 → 肝臓
  5. 肝臓 → 腸管 → 門脈 → 胆管 → 肝臓
  6. 肝臓 → 腸管 → 胆管 → 門脈 → 肝臓

　解答　　

4

　解説　　

腸肝循環とは、胆汁排泄された薬物が小腸で、再び吸収され肝臓へ運ばれる　という物質の循環です。『肝臓で抱合を受け胆汁中へ排泄される』のだから、薬物は肝臓→胆管→腸管　という順番の経路となります。

そして『小腸で又分解されて吸収を受ける』のだから、これは普通の吸収における肝初回通過効果の経路となるので、腸管→門脈→肝臓　という順番です。

以上より、正解は 4 です。
